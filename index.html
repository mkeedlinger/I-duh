Ido

Resources for learning and maintaining a knowledge of Ido, an artificial auxiliary language that aims to be easy to learn and consistent. (NOTE: Ido is a spoken language, not a computer language)

What is Ido?

Ido is an artificial language that was created to be a universal second language for speakers of diverse backgrounds. It was meant to be as easy to learn as possible, and be grammatically, orthographically, and lexicographically consistent and regular.

It was created in 1907 based off of Esperanto (another language which had identical goals) because of a desire to reform flaws that Esperanto had when it was created 20 years earlier.

FAQ

How is 'Ido' pronounced?

Ido is pronounced /ˈiːdoʊ/ using the IPA (International Phonetic Alphabet) for English.

For those of you who didn't previously know about the IPA or how to read it, ' signifies that the primary stress follows it, i: is a long 'e' sound as in 'seed', d is a 'd' sound as in 'dye', and oʊ is a long 'o' sound as in 'bode'. The / simply signifies the beginning and end of the use of IPA.

Simply, Ido is pronounced eedo.

What alphabet does Ido use?

It uses an English alphabet without any diacritics, so you can use it without altering your keyboard or installing any software.

The pronunciation of each letter is a little different though, and much much more consistent that English.
